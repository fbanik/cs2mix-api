Como configurar o servidor

Passo a passo:
1. Crie conta em render.com
2. Suba os arquivos do backend no GitHub
3. No Render: New → Web Service → seu repositório
4. Em Environment Variables adicione:
  STEAM_API_KEY = sua chave da Steam
5. Pegue a chave em: steamcommunity.com/dev/apikey
6. Cole a URL gerada pelo Render aqui
